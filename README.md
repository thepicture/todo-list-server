# todo-list-server

Серверная часть приложения todo-list-client

## Установка

```bash
npm i
```

## Запуск

### Без Docker

```bash
npm start
```

### С Docker

```bash
sudo docker compose up
```

## Роуты

`/api/auth` - JSON авторизация
